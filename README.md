# Diary App (ASP.NET Core MVC)

This is a beginner-level **ASP.NET Core MVC** learning project that implements
a simple **Diary Management System** with full **CRUD operations**.

The purpose of this project is to understand the MVC architecture,
Entity Framework Core, and database integration using SQL Server.
More features will be added as I continue learning.

---

## Features

- Create a diary entry  
- View all diary entries  
- Edit an existing diary entry  
- Delete a diary entry with confirmation  

---

## CRUD Operations Used

- **Create** – Add new diary entries  
- **Read** – Display all diary entries  
- **Update** – Edit existing entries  
- **Delete** – Remove entries from the database  

---

## Tech Stack

- ASP.NET Core MVC  
- Entity Framework Core  
- SQL Server  
- Razor Views  
- Bootstrap  

---

## Project Purpose

This project is part of my **learning journey in ASP.NET Core MVC**.
It focuses on understanding:
- MVC pattern
- Model binding
- Database operations with Entity Framework
- Proper CRUD workflow

This is a starting point, and the project will be enhanced with
additional features in the future.
